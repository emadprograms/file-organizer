## ✅ Requirements Coverage
1. **Epic: Architecture (ARCH)**
   - `ARCH-01`: Refactored `src/processing/pipeline.py` to separate grouping and routing into distinct pipeline steps.
2. **Epic: Resilience (RES)**
   - `RES-01`: Implemented atomic `RoutingStateManager` to enable graceful resumption of the routing phase after mid-execution failures, complete with checksum-based guardrails.
3. **Epic: Configuration (CFG)**
   - `CFG-01`: Upgraded the routing functions to propagate explicit `model` overrides down to the LLM client calls.
4. **Epic: Routing (ROUT)**
   - `ROUT-01`: Transitioned to Arabic folder mappings, eliminated arbitrary routing via constrained Prompt configurations, and implemented a double-verification flow for ambiguous files ("Others" / "Miscellaneous").
