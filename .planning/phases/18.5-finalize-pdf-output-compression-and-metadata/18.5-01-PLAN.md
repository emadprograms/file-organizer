# Phase 18.5-01: Finalize PDF Output, Compression, and Metadata

## Objective
Finalize the PDF processing by creating a `[house_id]_finalized.pdf` output. This file will be a compressed version of the original `[house_id]_categorized.pdf` with an embedded Table of Contents (Bookmarks) mapping each page to its assigned routing folder.

## Dependencies
- Phase 18 (Completed)
- `18.5-CONTEXT.md` (Context locked in)

## Implementation Steps

### 1. Add TOC to the Original PDF (`src/main.py`)
- In `src/main.py`'s `run_generation_pass`, after `FileOrganizer.organize(...)` completes and returns the `per_page` mapping.
- If not a `dry_run`:
  - Open the original `pdf_path` (`*_categorized.pdf`) using PyMuPDF (`fitz`).
  - Construct a Table of Contents (TOC) array: `toc = []`.
  - Iterate through the `per_page` list.
  - For each entry, construct the bookmark title: `title = entry.get("target_folder")`.
  - Append to TOC: `toc.append([1, title, entry["page_index"] + 1])`. (Level 1, Title, 1-based page number).
  - Apply the TOC to the PDF using `pdf_doc.set_toc(toc)`.
  - Save the modified PDF to a temporary file: `tmp_path = house_dir / f"{full_house_id}_finalized.tmp.pdf"`.
  - Close `pdf_doc`.

### 2. Apply Compression (`src/main.py`)
- Import `compress_pdf` from `src.pdf.compress`.
- Define `finalized_path = house_dir / f"{full_house_id}_finalized.pdf"`.
- Call `compress_pdf(str(tmp_path), str(finalized_path))` to optimize images and apply `garbage=4, deflate=True`.
- Ensure `tmp_path` is safely removed if it remains after `compress_pdf`.

### 3. Verification & Cleanup
- Verify that `extract_pdf_segment` still functions independently for the individual folders.
- Ensure the newly created `_finalized.pdf` is not accidentally moved into `.source_files` during the final cleanup loop.

## Verification Steps
1. Run a sample PDF through the pipeline.
2. Check that `[house_id]_finalized.pdf` is present in the output directory.
3. Open `[house_id]_finalized.pdf` in a PDF viewer and verify that a Bookmark/Outline pane appears with entries matching the `Tenant / Folder Name`. Clicking a bookmark should jump to the correct page.
4. Verify the file size of `[house_id]_finalized.pdf` is noticeably smaller than `[house_id]_categorized.pdf` without unacceptable visual distortion.
