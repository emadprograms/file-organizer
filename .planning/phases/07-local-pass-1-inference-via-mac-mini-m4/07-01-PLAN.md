# Phase 07: Local Pass 1 Inference via Mac Mini M4

## Objective
Drastically speed up Pass 1 Vision Extraction and completely bypass Google API rate limits by moving the primary extraction locally to the Mac Mini M4 using `Qwen2.5-VL-7B-Instruct` via LM Studio's OpenAI-compatible endpoint.

## Context
- **Strategy:** Local-first extraction via the `openai` client (pointing to `http://localhost:1234/v1`).
- **Fallback:** If local inference fails, hangs, or crashes, fallback exclusively to `gemini-4-26b` via the existing `genai` logic.
- **Cleanup:** `gemini-2.5-flash` and `gemini-4-31b` are completely retired from the pipeline and must be removed.

## Artifacts this phase produces
- **Dependencies:** `openai` added to `requirements.txt`.
- **Symbols:** 
  - `openai.OpenAI` client instance logic inside `GemmaClient.classify_page` (or initialized in `__init__`).
  - `test_local_inference_fallback` test function in `tests/test_llm.py`.
