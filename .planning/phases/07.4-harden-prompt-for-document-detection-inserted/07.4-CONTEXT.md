**Gathered:** 2026-06-25

<domain>
## Phase Boundary

This phase delivers an improved and hardened prompt for document detection, specifically focusing on utilizing specific Arabic subject patterns to allow the local LLM to guess the document type accurately. When the subject is missing, it falls back to a larger model (e.g., Gemma 4 26b).

</domain>

<decisions>
## Implementation Decisions

### Prompt Hardening and Document Detection
- **D-01:** Always analyze the subject of the letter first.
- **D-02:** When the subject is missing, the local LLM is bypassed, and a larger model (like Gemma 4 26b) should be used instead.
- **D-03:** Local LLM should be utilized for `personal details` since it already detects them perfectly.

### Amar Takhsees Patterns
- **D-04:** If the subject is exactly "الموضوع : الوحدات السكنية".
- **D-05:** If the subject mentions an extension of stay (e.g., تمديد الإقامة / السكن).
- **D-05b:** STRICT DEFINITION: Ensure the AI clearly understands that an 'amar takhsees' (allocation order) is an order from a higher authority claiming or telling to give the primary tenant a place to stay. Do not classify random documents as allocation orders unless they meet these criteria or the strict subject patterns.

### Maintenance Patterns
- **D-06:** If the sender or receiver is "إدارة الأشغال" (idara ashgal).
- **D-07:** If the document is a yellow paper with inspection details.
- **D-07b:** If the document mentions "inspection" anywhere, it is part of the maintenance section.

### House Modifications Patterns
- **D-08:** If the subject contains "طلب" (talab) and mentions modifying the house.

### Key Handover Form Patterns
- **D-09:** If the document contains "استمارة تسليم الوحدات السكنية التابعة لوزارة الداخلية".

### EWA Patterns
- **D-10:** If the document contains a meter number, such as "الموضوع: الوحدة السكنية رقم ( 508 ) طريق 4411 مجمع 944 سافرة" or "حاب ( 13/19239) قم الحس".

### Allowance Patterns
- **D-11:** If the subject is "الموضوع: وقف استقطاع بدل الانتفاع" (Allowance deduction will NOT have "30 bd" or "60 bd" written on it).

### Rent Deduction Patterns
- **D-12:** If the document mentions rent deduction (e.g., استقطاع الإيجار).
- **D-12b:** STRICT DEFINITION: Rent deduction letters will ALWAYS have "30 bd" or "60 bd" written on them. Use this to explicitly disambiguate from allowance deduction.

### Notifications Patterns
- **D-13:** If the document contains the word "إشعار" or "اشعار".

### Basic vs Personal Details Patterns
- **D-14:** `basic details` are strictly forms about the person. If it is not a form, it cannot be classified as basic details.
- **D-15:** `personal details` refers to pictures of identity cards, passports, and other non-form documents related to the person and his family. Anything related to a person and his family that is NOT a form goes into personal details.

### the agent's Discretion
None.
