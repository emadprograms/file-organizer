# Phase 19.1.1.1: Update YAML loader to check .source_files

## Goal
Update `yaml_loader.py` to check the `.source_files/` directory for the YAML configuration instead of the root `target_dir`, and update tests to reflect this change.

## Must Haves
- The YAML loader correctly looks for `{house_id}_tenants.yaml` inside `target_dir / ".source_files" /`.
- Missing `.source_files/` or YAML file should gracefully fallback to anchor logic without crashing.
- Tests in `test_tenant_config_yaml.py` are updated to create `.source_files/` directory.
