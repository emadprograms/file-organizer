# Plan 11-01: Folder Mapping and Direct Routing Infrastructure

## Objective
Establish a strict 1:1 mapping between English YAML categories and Arabic target folders, and implement direct Python routing for high-confidence categories to bypass LLM calls.

## Research & Analysis
- **English Categories:** Identified from `sample-config.yaml` (e.g., `BASIC_DETAILS`, `CONTRACT`, `EWA_LETTERS`, `INSPECTION_PICTURES`).
- **Target Folders:** Need to be finalized as Arabic names.
- **Current Logic:** `src/processing/routing/router.py` uses `SINGLE_MATCH` for some direct routing, but it's based on `CATEGORY_TO_FOLDERS` derived from `FOLDER_ROUTING`.

## Tasks

### 1. Define Arabic Folder Mapping
- Create a definitive mapping between the 13 English YAML categories and their corresponding Arabic folder names.
- **Proposed Mapping (to be verified):**
    - `BASIC_DETAILS` -> `بيانات أساسية`
    - `PERSONAL_DETAILS` -> `بيانات شخصية`
    - `AMAR_TAKHSEES` -> `أمر تخصيص`
    - `KEY_HANDOVER` -> `محضر تسليم مفتاح`
    - `CONTRACT` -> `عقود`
    - `EWA_LETTERS` -> `كهرباء وماء`
    - `RENT_DEDUCTION` -> `استقطاع إيجار`
    - `ALLOWANCE_DEDUCTION` -> `وقف استقطاع بدل`
    - `NOTIFICATIONS` -> `إشعارات`
    - `MAINTENANCE` -> `صيانة`
    - `INSPECTION_PICTURES` -> `صور ومعاينات`
    - `MODIFICATIONS` -> `تعديلات`
    - `OTHER_LETTERS` -> `رسائل متنوعة`
- Update `src/processing/routing/config.py` to replace the current `FOLDER_ROUTING` keys and values with this new mapping.

### 2. Implement Direct Python Routing
- Refactor `route_document` in `src/processing/routing/router.py` to explicitly bypass the LLM for:
    - `PERSONAL_DETAILS` (ID)
    - `CONTRACT` (Contract)
    - `EWA_LETTERS` (Utility Bills)
    - `INSPECTION_PICTURES` (Pictures)
- Ensure these categories return `(folder_name, True)` immediately.

### 3. Update Routing Configuration
- Ensure `CATEGORY_TO_FOLDERS` and `SINGLE_MATCH` are correctly derived from the new Arabic mapping in `config.py`.

## Verification Plan
- **Unit Test:** Add tests to `tests/test_routing.py` to verify that `PERSONAL_DETAILS`, `CONTRACT`, `EWA_LETTERS`, and `INSPECTION_PICTURES` are routed directly without calling the LLM.
- **Validation:** Verify that the returned folder names are the correct Arabic strings.
- **Log Check:** Confirm that routing decisions for these categories are logged as `is_direct_routed=True`.
