# Plan 02-05: Gap Closure - Robustify Name Extraction and Fix Sliding Window

## Objective
Fix the UAT gaps regarding inaccurate name extraction (frequent NONEs and fragmented names) and incorrect page grouping (continuation pages being split into separate documents).

## Steps

### Step 1: Fix Sliding Window Context in Pipeline
- **File:** `src/pipeline.py`
- **Action:** 
  - Update `classify_page` signature call to accept a new parameter `active_summary`.
  - Inside the main loop, dynamically maintain an `active_summary` string that describes the current group being built (e.g., `Active group: pages X-Y, category: Z, resident: A`).
  - Pass this `active_summary` along with `previous_summary` to `client.classify_page()`.
  - When a new group starts, set `previous_summary` to the finalized `active_summary`, and start a new `active_summary`.

### Step 2: Update LLM Context and System Prompt
- **File:** `src/llm.py`
- **Action:**
  - Update `classify_page` signature to accept `active_summary: str = ""`.
  - Incorporate `active_summary` into the `user_prompt` (e.g., `Currently active document group: {active_summary}. Is this page a continuation of it?`).
  - Enhance the `_build_system_prompt` rules:
    - Explicitly demand capturing the full 4-5 part Arabic name.
    - Mention that names can be in Arabic or English, and to prioritize the longest/most complete version.

### Step 3: Implement Name Extraction Retry Logic
- **File:** `src/llm.py`
- **Action:**
  - In `classify_page`, add a localized retry loop specifically for checking `NONE` names.
  - If the initial LLM response yields `resident == "NONE"` (and the category is not `amar_takhsees` or another category where `NONE` is expected), issue a direct follow-up prompt to the LLM (or just retry the whole call with an added prompt: `Warning: You previously returned NONE for the name, but there is likely a name on this document. Look extremely closely for any Arabic or English names.`).
  - Allow 1 retry for missing names to respect rate limits while improving robustness.
