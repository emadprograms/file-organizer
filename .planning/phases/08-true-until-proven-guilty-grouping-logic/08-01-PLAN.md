# Phase 8: "True Until Proven Guilty" Grouping Logic

## Overview
This phase introduces a hybrid grouping logic based on document categories, ensuring documents are assumed to be continuations ("True Until Proven Guilty") unless a definitive theme shift ("Hard Reset") occurs. It extracts prompts into a configuration file and introduces category-specific routing and chunking sizes.
