# Threat Model
- **STRIDE Analysis**: 
  - *Tampering*: Deleting or altering temporary folders manually during `propose()` could crash `finalize()`. Handled by checking existence of tmp_dir before proceeding in finalize.
  - *Denial of Service*: Appending to large `_finalized.pdf` might be memory intensive. PyMuPDF is efficient enough to handle hundreds of pages seamlessly. Orphaned tmp folders could fill up storage (handled by stale cleanup logic in process_inbox).
  - *Information Disclosure*: Writing JSONs to `.tmp_` in the inbox keeps intermediate data safe and hidden (dot folder).
  - *Elevation of Privilege*: Path traversal via malicious filenames is mitigated by strictly resolving IDs against directory contents.
