## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #000000 | Terminal Background |
| Secondary (30%) | #FFFFFF | Terminal Text |
| Accent (10%) | #00FF00 | Status keywords (`_Proposed`, ` OK`) |
| Destructive | #FF0000 | Errors only |

Accent reserved for: File names, status keywords (`_Proposed`, ` OK`, `_Failed`, `_Error`).

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Append ` OK` to filename |
| Empty state heading | N/A |
| Empty state body | Listening for new files in Inbox... |
| Error state | Append `_Failed` or `_Error` to filename + print user-friendly error to console |
| Destructive confirmation | Remove `_Proposed` from filename to reject and start over |
