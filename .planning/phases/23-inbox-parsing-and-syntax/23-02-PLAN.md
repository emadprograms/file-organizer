# Phase 23 - Plan 02: Inbox Data Inference and Resolution

## Requirements
- FSUI-02
- FSUI-03

## Goal
Build the resolution logic to infer missing 'U' data using majority-vote from `_report.json`, resolve area directory conflicts, and integrate these capabilities into the `append` mode CLI hook.

## must_haves
- U resolution uses majority vote on _report.json pages
- House area conflicts raise specific errors for renaming
- Tenant hint resolves via canonicalize_with_llm with the correct target_dir
- Append mode in main.py catches errors, initializes LLMClient, and renames file with specific suffixes
- categorization.py is refactored to support processing a single specific file
