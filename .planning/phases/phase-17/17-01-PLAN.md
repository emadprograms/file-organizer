# 17-01-PLAN: YAML Configuration Loader

## Objective
Implement a YAML loader function in the `src/tenant_config` module that can find and parse `tenants.yaml` from the execution target directory.

## Steps

1. **Create `src/tenant_config/yaml_loader.py`**
   - Import `yaml` and `pathlib.Path`.
   - Create a function `load_tenant_config(target_dir: Path) -> list[str]`.
   - The function should look for `tenants.yaml` in the given `target_dir`.
   - If the file exists, read and parse it using `yaml.safe_load()`.
   - Extract and return the list under the `tenants` key.
   - If the file is missing, log a warning and return an empty list.
   - If the file is malformed or the `tenants` key is missing, raise a `ConfigurationError`.

2. **Write Unit Tests**
   - Create a test file `tests/test_tenant_config_yaml.py`.
   - Write tests for:
     - Successfully loading a valid `tenants.yaml`.
     - Handling a missing file (returns `[]` and logs warning).
     - Handling malformed YAML or missing `tenants` key (raises `ConfigurationError`).
